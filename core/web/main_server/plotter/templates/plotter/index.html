План города представляет собой прямоугольник, разбитый двумя рядами параллельных улиц на N×M квадратных кварталов. Вертолет вылетает из одного угла города и летит по прямой в противоположный угол города. Определите количество кварталов, над которыми пролетит вертолет. Границы кварталов не относятся к кварталу, то есть считается, что вертолет пролетает над кварталом, если он пролетает над его внутренней точкой. Входные данные содержат два натуральных числа, не превосходящих 10^9 (миллиард) – количество кварталов в сторонах прямоугольного города. Программа должна вывести число – количество кварталов, над которыми пролетит вертолет. И ГРАФИЧЕСКИ отобразить путь вертолета над городом.
